**Badge** — small uppercase pill for product flags and status. Tones map to the site's accent colors; `onDark` is for hero/footer placement.

```jsx
<Badge tone="cyan">Populārs</Badge>
<Badge tone="sky">Dual SIM · 4G</Badge>
<Badge tone="onDark" dot>Live</Badge>
<Badge tone="danger">Trauksme</Badge>
```

Tones: `cyan` (default), `sky`, `indigo`, `soft`, `success`, `warning`, `danger`, `onDark`. Set `dot` for a leading status dot (live feeds).
