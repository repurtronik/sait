**Stat** — a headline metric: heavy 800-weight number above a small uppercase label. Used in the hero stats strip and the About page counters.

```jsx
<Stat value="10 000+" label="Aizsargāti objekti" />
<Stat value="24/7" label="Atbalsts" onDark />
```

Set `onDark` on navy/hero backgrounds (number turns cyan). Lay several out in a flex/grid row.
