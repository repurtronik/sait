**SectionHeading** — the eyebrow + title + subtitle stack that opens nearly every section on cortex.eu. The sky-blue uppercase label is a signature brand device.

```jsx
<SectionHeading
  label="Risinājumi"
  title="Visaptveroša drošības ekosistēma"
  subtitle="No viena objekta līdz valsts mēroga monitoringa tīklam."
/>

<SectionHeading label="Tehnoloģijas" title="UHF / VHF tīkls" onDark />
```

Set `onDark` on navy/tech sections. `align="center"` for centered intros (CTA blocks).
